int render_frame ( t_game * game)

Renders a complete frame of the game.

Parameters
gamePointer to game structure
Returns
int Always returns 0

Rendering order:

  1. Clear buffer
  2. Draw floor tiles
  3. Draw exit
  4. Draw walls
  5. Draw player with direction/animation
  6. Draw collectibles
  7. Draw UI elements
