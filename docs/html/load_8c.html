void load_assets ( t_game * game)

Loads all game assets from files.

Parameters
gamePointer to game structure

Loads and initializes:

  • Floor tiles (10 variants)
  • Player animations (idle and running)
  • Collectible items
  • Wall textures
  • Exit texture All textures are scaled to match tile_size (32x32)

Loads a sequence of sprite frames from files.

Parameters
mlxMLX instance pointer
arrArray to store the loaded sprites
base_pathBase path to the sprite files
framesNumber of frames to load

Loads a sequence of XPM files named f0.xpm through f{frames-1}.xpm from the specified directory. Each sprite is scaled to 32x32 pixels.
